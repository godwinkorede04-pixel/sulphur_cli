# Sulphur CLI
This launcher is a Command Line Interface(CLI) implementation of the  [Sulphur Core Library](https://github.com/demperor-music/sulphur-core).
You are expected have a machine with Linux, MacOS or any other Unix/Unix-like OS.

## Installation

On linux/MacOS, you can install this program by either:

* Downloading the binary from the latest release.
* Installing the program using the [the nix package manager](https://github.com/NixOS/nix) with the following command:
```zsh
nix profile install github:demperor-music/sulphur_cli
```
